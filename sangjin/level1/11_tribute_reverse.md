# 11. 3진법 변환

## 문제 설명
자연수 n이 매개변수로 주어집니다.   
n을 3진법 상에서 앞뒤로 뒤집은 후, 이를 다시 10진법으로 표현한 수를 return 하도록 solution 함수를 완성해주세요.

## 입출력 예
n|result
-|------
45|7
125|229


### 제한 조건
- n은 1 이상 100,000,000 이하인 자연수입니다.

## 변수 및 함수
### 매개변수
n: 변환할 10진법 수   

### 새로 정의 한 변수
arr: 3진법으로 변환한 수를 뒤집은 순으로 수가 들어갈 배열   
BASE: 변환할 진법 (3진법이기 때문에 3으로 고정)   
multi: 자릿수 별 제곱수    

## 풀이
주어진 수n을 M진법으로 변환 할 때 arr에 순서대로 저장하면 n을 M진법 상에서 뒤집은 수가 된다. 따라서 arr의 끝에서부터 다시 M진법을 10진법으로 변환하면 된다.
