# 7. 모의고사

## 입출력 예

### 입출력 예 #1
    answers: [1,2,3,4,5]

    result: [1]

### 입출력 예 #2
    answers: [1,2,3,4,5]

    result: [1]


### 제한 조건
- 시험은 최대 10,000 문제로 구성되어있습니다.
- 문제의 정답은 1, 2, 3, 4, 5중 하나입니다.
- 가장 높은 점수를 받은 사람이 여럿일 경우, return하는 값을 오름차순 정렬해주세요..

## 변수 및 함수
students: 수포자 별 찍는 방법이 들어있는 이차원 배열   
score: 각 수포자의 점수를 저장하는 배열   
max: 최고점   

## 풀이
1. 답과 학생이 제출한 답이 일치할 때 score를 올려준다.   
- 수포자가 찍는 방식의 사이클 길이보다 문제 수가 더 많을 수 있기 때문에 문제 번호 mod 사이클 길이를 해주어 반복하여 찍을 수 있게 함.
2. 각 수포자의 점수를 비교하여 최고점을 찾아냄.
3. 점수가 같을 경우 오름차순으로 넣어줘야 하기 때문에 앞에서부터 최고점과 비교하여 같은 점수인 수포자를 answer 배열에 넣어줌.
