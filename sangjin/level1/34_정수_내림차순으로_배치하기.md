# 34. 정수 내림차순으로 배치하기

## 문제 설명
함수 solution은 정수 n을 매개변수로 입력받습니다. n의 각 자릿수를 큰것부터 작은 순으로 정렬한 새로운 정수를 리턴해주세요. 예를들어 n이 118372면 873211을 리턴하면 됩니다.

## 입출력 예
n | return
--|-------
118372 | 873211

### 제한 조건
- `n`은 1이상 8000000000 이하인 자연수입니다.

## 변수 및 함수
### 매개변수
    n: 큰 것부터 작은 순으로 정렬하고 싶은 정수

### 새로 정의 한 변수
    arr: 각 자릿수의 숫자가 들어있는 배열

## 풀이
수를 자리수별로 저장 후 정렬, 오름차순으로 되어있기 때문에 reverse() 함수를 사용하여 반대로 해준 후 앞에서부터 순서대로 `(이전까지 계산된 수 * 10) + 현재 인덱스의 수`를 해주어 다시 정수로 만들어 준다.
