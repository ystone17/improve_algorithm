# 25. 수박수박수박수박수박수?

## 문제 설명
길이가 n이고, "수박수박수박수...."와 같은 패턴을 유지하는 문자열을 리턴하는 함수, solution을 완성하세요. 예를들어 n이 4이면 "수박수박"을 리턴하고 3이라면 "수박수"를 리턴하면 됩니다.

## 입출력 예
n | return
--|-------
3 | "수박수"
4 | "수박수박"

### 제한 조건
- n은 길이 10,000이하인 자연수입니다.

## 변수 및 함수
### 매개변수
    n: 반복할 횟수

## 풀이
`수박`이 두 글자이기 때문에 길이가 n일 때 `수박`이라는 단어는 n/2번 들어가고 n이 홀수일 경우에만 마지막에 `수` 글자가 추가된다.
