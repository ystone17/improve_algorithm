# 35. 정수 제곱근 판별

## 문제 설명
임의의 양의 정수 n에 대해, n이 어떤 양의 정수 x의 제곱인지 아닌지 판단하려 합니다.
n이 양의 정수 x의 제곱이라면 x+1의 제곱을 리턴하고, n이 양의 정수 x의 제곱이 아니라면 -1을 리턴하는 함수를 완성하세요.

## 입출력 예
n | return
--|-------
121 | 144
3 | -1

### 제한 조건
- n은 1이상, 50000000000000 이하인 양의 정수입니다.

## 변수 및 함수
### 매개변수
    n: 제곱근인지 아닌지 알고 싶은 정수

### 새로 정의 한 변수
    square: 해당 인덱스의 제곱을 원소로 갖는 배열

## 풀이
1. 제곱수가 50000000000000이하인 모든 수를 square 배열에 넣는다.
2. n의 양의 제곱근을 구한 후 `int`형으로 형변환하여 나머지를 버린다.
3. `2.`번에서 나온 정수가 `N`이라 할 때 `square[N]`과 `n`을 비교하여 같으면 양의 정수의 제곱근이고 다르면 양의 정수의 제곱근이 아니다.
