```python3
# 2016년 1월 1일은 금요일이다. 그리고 윤년

DAYS = ['FRI', 'SAT', 'SUN', 'MON', 'TUE', 'WED', 'THU']

MONTH_DAYS = [31, 29, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31]

def solution(a, b):
    days = -1
    if a != 1:
        for idx in range(1 ,a):
            days += MONTH_DAYS[idx-1]
    days += b
    answer = DAYS[days % 7]
    return answer

```

자 1년의 각 달은 보통 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31 이렇게 있지만
윤년의 경우 2월이 29일이다.

그루구 요일을 구하는거는 `so easy`한데, 어짜피 요일이 7개가 반복이니까, a, b의 입력으로 들어온 `월`과 `일` 기반으로
1월 1일로부터 며칠인지 구한뒤에 7로 나눠서 요일을 구한다.

2016년 1월 1일은 금요일부터 시작이여서 리스트의 첫번째도 금요일로 했다
