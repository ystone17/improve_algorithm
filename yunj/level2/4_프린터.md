
# 4_프린터
```
import java.util.*;

class Solution {
    public int solution(int[] priorities, int location) {
        int answer = 0;

        Deque<page> printer = new LinkedList<>();
        PriorityQueue<Integer> pq = new PriorityQueue<>(Collections.reverseOrder());

        for (int i = 0; i< priorities.length  ; i++)  {
            printer.add(new page(i, priorities[i]));
            pq.add(priorities[i]);
        }
        int count = 1;
        while(!printer.isEmpty()){
            page tmp = printer.pollFirst();
            if (tmp.priority == pq.peek()){
                if (tmp.index == location){
                    return count;
                } else{
                    pq.poll();
                    count++;
                }
            } else{
                printer.addLast(tmp);
            }
        }




        return answer;
    }
}

class page implements Comparable<page> {
    int index;
    int priority;

    page(int index, int priority) {
        this.index = index;
        this.priority = priority;
    }

    @Override
    public int compareTo(page p) {
        if (this.priority != p.priority){
            return this.priority < p.priority ? 1 : -1;
        } else{
            return 1;
        }
    }
}
```
---
우선순위와 입력받은 프린트 백로그에서의 인덱스를 가지는 page class 생성

page 를 요소로 하는 deque printer 와 우선순위를 정렬하기 위해 우선 순위 queue pq 를 생성한다.

printer deque를 순회하면서 요소 page의 priority와 pq에서 가장 큰 값을 비교하여 같으면 요소 page를 dequeue 하고, pq에서 poll해준다.  만약 주어진 location과 page의 index가 같으면 그때의 count를 반환한다.

page와 pq에서 가장 큰 값이 다를 경우 
요소 page를 addlast 해주고 count를 증가시킨다.
