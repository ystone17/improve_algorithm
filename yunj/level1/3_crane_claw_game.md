# 3_crane_claw_game
```
board:
[
[0,0,0,0,0], 5층 0
[0,0,1,0,3], 4층 1 
[0,2,5,0,1], 3층 2
[4,2,4,4,2], 2층 3
[3,5,1,3,1]  1층 4
]

moves : 
[1,5,3,5,1,2,1,4]

high :
[2,3,4,2,4]

bucket : 
[4,3,1,1....]
```
high배열은 board에서 각 세로 줄의 인형이 있는 최고층을 기록한 것이다.

checkHigh(), pick(), removePair() 세 함수로 작성하였다.

# checkHigh() :  
매번 board 이중배열을 탐색하는게 부담되어 최고 층수를 기록하기 위한 함수이다. 이해하기 쉽게 1층부터 5층으로 기록하고 인형이 없는 줄은 0층이다.

# pick(): 
high 배열을 이용해 moves배열에 따라 지워야 할 인형의 위치를 바로 찾아 인형을 0으로 바꿔준다.
high 배열의 정보도 업데이트 해주며, 0층 일 경우 지나간다.

# removeFair():

매 pick 함수 실행 후 removeFair 함수를 실행해야 테트리스 마냥 연쇄적으로 인형이 사라지는 변수를 차단할 수 있다.
