
# 10_Year_of_2016
```java
class Solution {
    public String solution(int a, int b) {
        String answer = "";
        
        int[] month = {31,29,31,30,31,30,31,31,30,31,30,31};
        String[] DaysOfTheWeek = {"FRI","SAT","SUN","MON","TUE","WED","THU"};
        
        int m = a - 1 ;
        int d = b - 1 ;
        
        int total = 0;
        
        for(int i =0; i< m ; i++){
            total += month[i];
        }
        
        total += d;
        
        answer = DaysOfTheWeek[total % 7];
        
        
        return answer;
    }
}
```
# 설명
1월 1일부터 몇일 차이나는지 계산 한뒤 7로 나머지 연산하여 요일을 구한다.
