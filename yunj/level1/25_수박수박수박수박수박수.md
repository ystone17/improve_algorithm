
# 24_수박수박수박수박수박수
```
class Solution {
    public String solution(int n) {
        String repeat = "수박";
        String odd = "수";
        StringBuffer sb = new StringBuffer();

        int r = n / 2;
        int o = n % 2;

        for (int i= 0; i< r; i++){
            sb.append(repeat);
        }
        if (o == 1){
            sb.append(odd);
        }
        return sb.toString();
    
    }
}
```

## StringBuffer 
String class 는
private final char[] value 배열에 문자열을 저장한다.
따라서 잦은 문자열 편집은 heap을 낭비시킨다.

StringBuffer는 memory를 참조하여 편집 할 수 있다.
